下面的代码对iris数据集进行分片，并对每一个分片的前两列数据进行排序，其中lapply函数可以将一个函数应用到一个列表

```
> speciesData <- split(iris,iris$Species)
> speciesData
> orderdata <- lapply(speciesData,function(x){x[order(x[,1],x[,2]),]})
> orderdata
```
